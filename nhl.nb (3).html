Ranking TOP 10 Players From 2019-2020 season Using Linear Regression

Using their points per game from the last four seasons from the top point producers in the 2019-2020 NHL season, we will use linear regression to see how they perform this year.

Leon Draisaitl:

points <- c(0.939, 0.897, 1.280, 1.549)
season <- c(1, 2, 3, 4)
leon <- data.frame(season, points)
# plot a scatter plot
plot(leon$season, leon$points)
# simple linear regression
r <- lm(points~season, data=leon)
# add the regression line
abline(r)

# predict points per game for the fifth season
 ld = predict(r, list(season=5))
 ld
     1 
1.7195

We can expect Leon Draisaitl to have a Points per game of 1.7195 this season.
